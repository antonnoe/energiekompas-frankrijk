## Energiekompas Frankrijk (MVP)

Doel: een extreem eenvoudige energie- en kosteninschatting voor kopers en eigenaren, met “wow” scenario-vergelijking.

### Run (lokaal)
Open `index.html` direct in de browser.

### Wat zit er nu in (MVP)
- Context eerst: adres (BAN autocomplete) of regio/klimaatzone
- Snelle invoer: m², bouwperiode, aanwezigheidprofiel, 1 totale verbruiksslider, hoofdverwarming
- Resultaat: kosten + grootste verbeterknop
- Scenario-sprong (wow): Baseline vs Scenario
